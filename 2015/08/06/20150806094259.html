Win10系统快捷键收集

其实Windows对于我来说每个版本的变化影响其实不是很大，为什么这么说呢，因为我的一些常用操作都是用快捷完成的，包括从Win7到Win8的开始菜单的取消之类的，由于以前用mac所以，对于应用选择之类的界面都是用快捷完成的，影响所以就不是很大的，用起来还会很直观和爽快，开始菜单列表的话有时候还得去找应用。

好了，废话不多说了，下面主要说说本人使用Windows以来日常使用的一些快捷键分享。

基础的快捷键，这类就是大家耳熟能详的快捷键Ctrl + C、Ctrl + V之类的就不在重复介绍了，主要说说在系统应用方面比较喜欢的快捷键操作：


  • Win + Q ：打开系统搜索，这个很常用，有时候需要查找什么，就是使用这个搜索东西，升级到Win10后可以打开搜索助手来完成操作。

  • Win + E : 这个就是老生常谈了，打开系统的资源管理器，其实简单的就是打开我的电脑。

  • Win + R ：这个也是常用的，打开系统的运行窗口，可以来执行系统命令。

  • Win + I ：打开系统设置界面，从win8开始，系统中好像存在有两个版本的控制面板功能，以前常用的是控制面板，现在是设置。

  • Win + P ：这个也很常用，平时演示投影可以使用得到。

  • Win + Tab ：这个大家应该都很熟悉的了，日常我们切换应用常用得到，WIn10之后可以进入到新的切换界面、

  • Win + D ：在显示桌面和应用之间切换，如果有应用前置，按下后则会直接显示桌面，也很常用，有时候需要从桌面上打开东西，直接切换到桌面。

  • Win + L ：直接锁屏，平时暂离电脑前面可以使用锁屏。

  • Win + X ：是我使用频率最高的一个按键，功能很强大，谁用谁知道。


上面一些就是日常使用的一些常用组合键，有很多没有加上去，太多了。使用Windows一直使用dexpot来完成工作区功能，现在Win10自带了多桌面的功能还是很喜欢的。


  • Win + Ctrl + D ：创建一个新的桌面

  • Win + Ctrl + F4：关闭当前的桌面

  • Win + Ctrl + ←或→ ：在不同的桌面之间切换


还是很方便的。

应用贴边的操作也是非常的棒的。


  • Win + ←或→ 或↑或↓：可以改变窗体的贴边，还是非常的方便。


，好用的快捷键还有非常的多，大家无需记住，只需要平时常用就会慢慢熟悉的，下面贴一下一个快捷键的表，供大家参考。

任务视图：Win + Tab(松开键盘界面不会消失)

关闭当前虚拟桌面：Win + Ctrl + F4
切换虚拟桌面：Win + Ctrl +左/右
